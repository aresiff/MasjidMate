# MasjidMate - Laman Web Sistem Pengurusan Sukarelawan (Kod Program: CSC2764)
Membangunkan sebuah sistem berasaskan web untuk mengurus acara masjid dan aktiviti sukarelawan, yang menampilkan capaian berasaskan peranan bagi Pentadbir, Jawatankuasa dan Sukarelawan. Sistem ini menyokong penciptaan acara, permohonan sukarelawan, aliran kerja kelulusan, penjejakan prestasi serta penjanaan laporan.

Alat & Teknologi: Django (Python), HTML, CSS, SQLite, Rajah UML, Operasi CRUD
